# RespTable

## Installing example via npm in root folder
```
npm install
npm run build
```
## Copy files from new dist folder to example
```
cp dist/* examples/react-example/src/compiled-files-here/
```
## Build the example
```
cd examples/react-example/src/

npm install
```
## Start the example
```
npm start
```
## Access the example

Default port on localhost is 3000.
